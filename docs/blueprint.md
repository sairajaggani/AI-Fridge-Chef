# **App Name**: FridgeChef

## Core Features:

- Ingredient Input: Users can input a list of ingredients they have available.
- Recipe Generation: Generate recipe suggestions based on the available ingredients using AI.
- Recipe Display: Display generated recipes with clear instructions and ingredient lists.
- Recipe Filtering: Allow users to filter recipes based on dietary restrictions or cuisine preferences.

## Style Guidelines:

- Primary color: Fresh green (#A7D1AB), symbolizing freshness and health, aligning with the culinary theme.
- Background color: Light, desaturated green (#E8F2EC), providing a soft and clean backdrop.
- Accent color: Muted yellow (#D4AC0D), complementing the green to highlight important elements and actions.
- Body and headline font: 'PT Sans', a versatile humanist sans-serif suitable for both headlines and body text.
- Use minimalist icons to represent ingredients and recipe categories.
- A clean, card-based layout to showcase recipes and ingredients effectively.
